# SIO Breakout Board

This contains a SIO Socket Connector and a 13 pin header connector.  
It is designed to allow a SIO cable to be plugged into the socket,
and the header plugged into a breadboard, for easy prototyping of SIO peripherals.

SIO Connector Part taken from the following boards:

https://github.com/marekl123/atari-sio-connector

https://github.com/pmandes/atari-sio-connector
